## Usage

1. Download latest build from Relases
2. Install Melonloader and put the dll in ``Mods``
3. Run Miside Zero, when you begin the game ``Mods/mapper/nodes.json`` will be created

## Building

 1. Clone this repo
 2. Change the refrences to the ones dumped by Melonloader's Cpp2Il
 3. Build and use
